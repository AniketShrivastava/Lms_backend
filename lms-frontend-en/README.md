#LMS Frontend

### Setup instructions

1. Clone the project

...
    git clone https://github.com/AniketShrivastava/Lms_backend/tree/main/lms-frontend-en
...

2. Move into the directory

...
   cd Lms-frontend-en
...

3. Install dependancies

...
    npm install
...

4. Run the server

...
   npm run dev
...
